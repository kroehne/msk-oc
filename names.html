Übersicht

Baustein Auflage Version Runtime Project
B1A, B1B, …, S4B edition-1 oder edition-2 V01 oder V02 9.9.0 oder 10.0.0 *_1.zip, *_2.zip, …

Anmerkungen:

  • CBA-ItemBuilderProjektdateien in unterschiedlichen Runtime-Versionen sind inhaltsäquivalent.
  • Version identifiziert die inhaltliche Version eines Bausteins (gleiches Fördermaterial)
  • Auflage identifiziert die konzeptuell veränderten Bausteine (anderes Fördermaterial)
